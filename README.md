# Organizador de Arquivos

Projeto em Python para analisar e organizar arquivos automaticamente.

## Funcionalidades

- Escaneamento de diretórios
- Classificação de arquivos
- Detecção de arquivos duplicados
- Geração de relatórios

## Estrutura

```text
organizador/
├── scanner.py
├── classifier.py
├── duplicates.py
```

## Tecnologias

- Python
- pathlib
- hashlib
- GitHub
